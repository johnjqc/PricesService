# ADR 0004 - Use Spring Boot

## Status
Accepted

## Context

The project requires a mature framework for building REST APIs
and integrating infrastructure components.

## Decision

Use Spring Boot as the main application framework.

Benefits:

- Dependency injection
- REST support
- ecosystem integration
- production readiness

## Consequences

### Positive

- Rapid development
- Mature ecosystem
- Production-ready tooling

### Negative

- Framework lock-in
